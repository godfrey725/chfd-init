# Local Development Runbook

Document how to run the project locally.

## Include
- prerequisites
- install steps
- environment setup
- main development commands
- how to run tests or checks locally
- common reset or recovery steps for local state

## Expectations
- Prefer exact commands over vague descriptions.
- Call out common setup pitfalls.
- Keep this file updated when local workflows change.
- Note which commands are the fastest path for daily development.
